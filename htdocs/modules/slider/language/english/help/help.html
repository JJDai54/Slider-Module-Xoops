Help: slider - Slides management

Description

This module allows you to manage the slides of the home page of the themes used
It only works 'with themes of type "xbootstrap".
The file " tpl / slider.tpl " must exist.
It may be necessary to write it to allow its modification.

Features :

  • Adding, Deleting slides to the home page
  • Activation, deactivation of slides, without deletion
  • Defined the display type: with or without display period
  • Definition of the display order of the slides
  • Definition of a display period to anticipate an event ( recurring parties for example)


Install / uninstall

None special measure is needed, follow the process standard installation and extract the cursor folder into the ../modules directory. Install the module through Admin -> Module system -> Modules.
Detailed instructions on installing modules are available at XOOPS Operations Manual

Tutorial

Once installed, actions to perform:

  • Standard installation & nbsp; module
  • Add slides to the themes used (see "Adding a new slide)" "
  • Activate the module block, regardless of its place it will remain invisible
    Important : the "Show block" option of the block Lists the selected slides

Addition of a new slide

Title: message displayed on the slide.
Description: Message displayed under the title on the slide.
Image: Image of the slide. attention check the file size (1920px x 500px for xbootstrap).
Active : no = The slide will never be displayed. Allows you to activate a slide for later use without deleting it, for example for recurring parties.
Visible: yes = the slide is always displayed - & nbsp; no = the display period must be defined.
Start date: date and time of the start of slide display.
End date: end date of slide display.
Weight : define the display order of the slides. If several slides have the same weight the alphabetical order of the titles is applied.

Treatments

On the index page several treatments are & nbsp; possible.

  • Reset the original 'tpl / slider.tpl' files of each theme.
    Disable the slider module block to allow the display of the slides of origin
    Remove the caches to update the display immediately


  • Activate the block which allows updating slides
    Equivalent to displaying the block in the block manager


Block -> slider_update_tpl

When the block is activated, the block modifies the file "tpl / slider.tpl" with the slides that have been added.

The update is only carried out if one of the slides has been modified, added or deleted.
The original file is saved with the suffix "-old" -> "tlp / slider-old.tpl"
It will be reinstalled when uninstalling the module.
