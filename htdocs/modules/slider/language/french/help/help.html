Help: slider - Slides management

Description

Ce module permet de gérer les slides de la page d'accueil des thèmes utilisés
Il ne fonctionne qu'avec les thème de type "xbootstrap".
Le fichier "tpl/slider.tpl" doit exister.
Il sera peut être nécéssaire de la mettre en écriture pour permettre sa modification.

Fonctionalités :

  • Ajout, Suppression de slides à la page d'accueil
  • Activation, désactivation des slide, sans supression
  • Défini le type d'affichage: avec ou sans période d'affichage
  • Définition de l'ordre d'affichage des slides
  • Définition d'une période d'affichage pour anticiper un événnement (fêtes récurrentes par exemple)


Install/uninstall

Aucune mesure particulière n'est nécessaire, suivez le processus d'installation standard et extrayez le dossier du curseur dans le répertoire ../modules. Installez le module via Admin -> Module système -> Modules.
Des instructions détaillées sur l'installation des modules sont disponibles dans le XOOPS Operations Manual

Tutorial

Une fois installé, actions à effectuer :

  • Installation standard  de module
  • Ajouter des slides au thèmes utilisés (voir "Ajout d'un nouveau slide)""
  • Activer le block du module, peu importe sa place il restera invisible
    Important : l'option "Afficher le block" du block Permet de lister les slides sélectionnés
                  à utiliser en mode débugage uniquement
                  laisser la sur "Non" en production

Ajout d'un nouveau slide

Titre : message affiché sur le slide.
Description : Message affiché sous le titre sur le slide.
Image : Image du slide. attention vérfiez la taille des fichiers (1920 px x 500 px pour xbootstrap).
Actif : non = Le slide ne sera jamais afficher. Permet de sactiver un slide pour une utilisation ultérieure sans le supprimer, par exemple pour les fêtes récurrentes.
Visible : oui = le slide est toujours affiché -  non = il faut définir la pariode d'affichage.
Date de début : date et heure de début d'affichage du slide.
Date de fin : date de fin d'affichage du slide.
Poids : défini l'ordre d'affichage des slides. Si plusieurs slides ont le même poids l'ordre alphabétique des titres est appliqué.

Traitements

Sur la page d'index plusieurs traitements sont  possibles.

  • Rinitialise les fichiers 'tpl/slider.tpl' d'origine de chaque thème.
    Désactive le block du module slider pour permettre l'affichage des slides d'origine
    Supprime les caches pour actualiser l'affichage immédiatement


  • Activer le block qui permet la mise à jour des slides
    Equivaut à afficher le block dans le gestionnaire de blocks


Block -> slider_update_tpl

Quand le block est activé, le block modifie le fichier "tpl/slider.tpl" avec les slides qui ont été ajoutés.

La mise à jour n'est effectué que dans le cas un des slides a éété modifier,ajouter ou supprimé.
Le fichier d'origine est sauvegardé avec le le suffixe "-old" -> "tlp/slider-old.tpl"
Il sera réinstallé lors de la désinstllation du module.
